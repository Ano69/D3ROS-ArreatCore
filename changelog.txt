Changelog : 

- Version 1.3 : 17/04/2014

- Ajout de nouvelles séquences bounty
- Ajout de nouvelles fonctions dans les séquences (ifposition / loadsequence / positionrange)
- Gestion de grablistXP pour toute les difficultés
- Modification de la configuration par défaut pour des runs plus rapides
- Légère modification du sequencer pour des images plus faciles a corréler
- Fix gestion Cursed Chest
- Fix conflit grablist et configuration
- Fix gestion de l'onglet repair entre les actes
- Fix gestionmenu pour lag et machine lentes
- Suppression des decorlist dans les séquences par defaut
- Ajout d'une aide sur les commandes des séquences (_aide_sequence.txt)

- Version 1.2 : 15/04/2014

- Mise à jour de l'icône du bot
- Ajout d'une interface graphique en mode Devmode=True
- Ajout d'un sequencer fonctionnel avec un interface graphique (Sequencer.exe)

- Version 1.1 : 15/04/2014

- Ajout d'une nouvelle option : NoBountyFailbackToAdventure
- Correction d'un bug en mode farming de boss
- Amélioration de la lisibilité du settings.ini
- Par défaut le bot est maintenant en mode semi-automatique et run Aventure
- Augmentation des temps pour les loots en fin de séquence boss

- Version 1.0 : 14/04/2014

1ère version publique du merge de l'ancien Bot Automatisation 8.1 et de la version 8.2i avec de nombreux ajouts

- Mode automatique / ou manuel ou semi automatique permettant :
* Des runs dans n'importe quel mode et sur n'importe quel act (Avec rotations aléatoires, ...)
* Des runs en mode aventure ou en mode campagne
* Des runs en mode bounty (Lancement automatique des séquences pour les bounty en cours)
* Une sélection automatique de la difficulté du héros ou des quêtes

- Une très grande selection de séquences pour couvrir le maximum de besoins.

- Une gestion de settings par héros :
* permet de séparer les configuration des héros pour ne pas modifier un fichier lorsque l'on change de personnage

- De nombreuse nouvelles options : (Voir les fichiers settings pour plus d'aide)
* ChaseElite pour pourchaser les élites
* PrioriyMonsterlist pour pourchaser certains mobs comme les gobelins
* Gestion des pauses
* Options pour une gestion complète du recyclage, de la revente et des grablists
* Gestion des listes de coffres / racks / decors a ouvrir / casser
* Gestion de différentes grablists
* Mode debug pour génération facile de logs pour le support
* Gestion de la sécurité en hardcore
* Configuration des touches pour l'ensemble des fonctions
* Gestion dynamique des timeouts

Et surement beaucoup plus
